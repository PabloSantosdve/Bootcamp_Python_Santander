# Bootcamp Python – DIO & Santander

Este repositório reúne **todos os exercícios desenvolvidos ao longo do Bootcamp de Python da Digital Innovation One (DIO), em parceria com o Santander**.

O objetivo é consolidar os fundamentos da linguagem Python por meio da prática contínua, cobrindo desde conceitos básicos até tópicos mais avançados.

---

## Conteúdos Abordados

Ao longo do bootcamp, foram estudados e praticados os seguintes temas:

- Variáveis e tipos de dados  
- Operadores  
- Estruturas condicionais e de repetição  
- Estruturas de dados e listas  
- Strings e fatiamento  
- Manipulação de arquivos  
- Data e hora  
- Decoradores, iteradores e geradores  
- Programação Orientada a Objetos (POO)  
- Gerenciamento de pacotes e boas práticas  
- Controle de versão com Git  

---

## Estrutura do Projeto

O repositório está organizado por **tópicos de estudo**, conforme a progressão do Bootcamp.  
Cada diretório representa um módulo específico, contendo exercícios práticos relacionados ao tema abordado.

### Organização dos Diretórios

- **conhecendo a linguagem/**  
  Conceitos iniciais da linguagem Python e primeiros comandos.

- **variaveis/**  
  Declaração, atribuição e uso de variáveis.

- **tipos/**  
  Tipos de dados primitivos e conversões.

- **operadores/**  
  Operadores aritméticos, relacionais e lógicos.

- **Estruturas condicionais e de repetição/**  
  Uso de `if`, `else`, `elif`, `for` e `while`.

- **estruturas de dados e listas/**  
  Manipulação de listas, tuplas, dicionários e conjuntos.

- **Dominando String e Fatiamento/**  
  Operações com strings e técnicas de fatiamento.

- **Manipulação de arquivos/**  
  Leitura e escrita de arquivos.

- **Data e Hora/**  
  Trabalho com datas, horários e formatação.

- **Decoradores, Iteradores e Geradores/**  
  Conceitos avançados da linguagem.

- **POO (Programação Orientada a Objetos)/**  
  Classes, objetos, métodos e encapsulamento.

- **Gerenciamento de pacotes e boas praticas/**  
  Uso de dependências, organização de projetos e boas práticas.

- **git/**  
  Conceitos e práticas de versionamento de código.

### Arquivos de Configuração

- **Pipfile / Pipfile.lock**  
  Gerenciamento de dependências com Pipenv.

- **.gitignore**  
  Definição de arquivos ignorados pelo controle de versão.

- **.vscode/**  
  Configurações do ambiente de desenvolvimento.

Cada diretório contém exercícios práticos relacionados ao respectivo assunto.
